# Poppy — character sheet + ingredient image
> Poppy is one half of the locked recurring couple (see CONCEPT.md §2). Render these ONCE and reuse
> across every episode. The **design sheet** (portrait + full body) is for human review and for
> anchoring the look when rendering STORYBOARDS in Imagen. The **ingredient image** (clean single) is
> what you feed **Veo's "Ingredients to Video"** mode for video generation (a multi-panel sheet reads
> as a busy grid and won't transfer — see CONCEPT.md §7).
> The SHARED STYLE CLAUSE below is identical in Bruno's file — keep it verbatim in both so the two
> characters render in exactly the same style.

## How to use
- Render `poppy-sheet.jpg` for design lock + as an Imagen reference when rendering storyboards.
- Render `poppy-ingredient.jpg` and use it as the Poppy *ingredient* in Flow's "Ingredients to Video"
  mode (alongside `bruno-ingredient.jpg` and a setting still) for every scene — see CONCEPT.md §7.
- Always restate Poppy's traits in prompt text too (Flow is stateless): small cheeky cream-white
  bunny, buck teeth, cocked eyebrow, sly grin, long floppy ears, blush-pink sweater.
