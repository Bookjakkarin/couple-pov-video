# Bruno — character sheet + ingredient image
> Bruno is one half of the locked recurring couple (see CONCEPT.md §2). Render these ONCE and reuse
> across every episode. The **design sheet** (portrait + full body) is for human review and for
> anchoring the look when rendering STORYBOARDS in Imagen. The **ingredient image** (clean single) is
> what you feed **Veo's "Ingredients to Video"** mode for video generation (a multi-panel sheet reads
> as a busy grid and won't transfer — see CONCEPT.md §7).
> The SHARED STYLE CLAUSE below is identical in Poppy's file — keep it verbatim in both so the two
> characters render in exactly the same style.

## How to use
- Render `bruno-sheet.jpg` for design lock + as an Imagen reference when rendering storyboards.
- Render `bruno-ingredient.jpg` and use it as the Bruno *ingredient* in Flow's "Ingredients to Video"
  mode (alongside `poppy-ingredient.jpg` and a setting still) for every scene — see CONCEPT.md §7.
- Always restate Bruno's traits in prompt text too (Flow is stateless): big goofy caramel-brown bear,
  cowlick tuft, tongue-out grin, round belly, too-small forest-green hoodie.
